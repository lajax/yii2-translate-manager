a() public method

Translating one dimensional array.

e.g.: $array = [

 'hello' => 'Hello {name}!',
 'hi' => 'Hi {name}',

];

$params = [

 'hello' => [
     'name' => 'World,
 ],
 'hi' => [
     'name' => 'Jenny',
 ],

]; $result = \lajax\translatemanager\helpers\Language::a($array, $params);

The result:

[ 'hello' => 'Hello World', 'hi' => 'Hi Jenny', ]

d() public method

For translating language elements stored in a database.

Enable translating databases in config before use. e.g.: 'modules' => [

 'translatemanager' => [
     'class' => 'lajax\translatemanager\Module',
     'tables' => [
         [
             'connection' => 'db',
             'table' => 'language',
             'columns' => ['name', 'name_ascii'],
         ],
         [
             'connection' => 'db',
             'table' => 'product',
             'columns' => ['name', 'description'],
         ]
     ]
 ]

]
